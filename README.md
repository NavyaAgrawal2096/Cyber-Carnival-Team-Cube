# DontClickThat

*A Cybersecurity Awareness Web Application*

##  Project Overview

**DontClickThat** is a cybersecurity awareness web application developed as part of the **Cyber Carnival Hackathon** by **Team Cube**.

The platform is designed to educate users about malicious URLs, phishing attacks, and unsafe browsing practices. It provides an interactive interface where users can learn how to identify suspicious links and test their understanding through guided content.

The primary objective of this project is to promote safe online behavior and build foundational awareness about common cyber threats.

---

##  Objectives

* Raise awareness about phishing and malicious URLs
* Provide educational content on identifying unsafe links
* Offer a simple interface for URL checking simulation
* Encourage safe browsing habits among users

---

##  Features

* Informative learning module on malicious links
* Interactive interface for URL safety checking
* Educational quiz section to reinforce concepts
* Clean and user friendly front end design
* Lightweight and easy to deploy
* Toggle between light and dark mode

---

##  Technical Approach

* **HTML5** – Structure and layout
* **CSS3** – Styling and responsive design
* **JavaScript** – Interactive functionality and URL validation logic
